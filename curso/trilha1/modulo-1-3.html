⭐ As Melhores para Começar (Starter Pack)

As skills canônicas que todo mundo deveria conhecer e instalar primeiro. Oito nomes que ensinam o ecossistema mais rápido do que qualquer tutorial — e o kit mínimo para sair instalando hoje.

🧭 Por que existe um "starter pack"

Com 39.366 skills no catálogo, ninguém começa pela cauda longa. Você começa pelas skills que são, ao mesmo tempo, muito instaladas (sinal de que funcionam), bem escritas (servem de modelo) e de fontes confiáveis (vercel-labs, anthropics, supabase). Instalar e ler essas skills é o atalho mais rápido para entender o que uma skill boa parece por dentro.

✗ Começar errado

  • ✗Sair instalando skills aleatórias da cauda longa
  • ✗Copiar uma skill com 3 installs como "modelo"
  • ✗Tentar criar a sua antes de ler uma boa

✓ Começar pelo pack

  • ✓Instalar 4-8 skills canônicas e ler o SKILL.md delas
  • ✓Ver como players sérios escrevem description e corpo
  • ✓Ganhar utilidade imediata enquanto aprende o formato

💡 Instalar é ler de graça

Cada skill que você instala vira um arquivo SKILL.md no seu projeto, em skills/ ou .claude/skills/. Abra esses arquivos: o starter pack é também a melhor coleção de exemplos comentados que você vai achar.

2

🔎 A indispensável: find-skills

Se você instalar só uma, instale esta. find-skills (vercel-labs/skills) é a skill mais instalada do planeta — 1.802.925 installs — e faz uma coisa só, perfeitamente: ensina o agente a buscar, avaliar e instalar outras skills do catálogo sozinho. É a meta-skill que destrava todas as outras.

find-skills

1.802.925 installs

O que faz: dá ao agente a habilidade de pesquisar o skills.sh por linguagem natural, comparar candidatas e instalar a melhor — sem você precisar abrir o site.

Por que entra no pack: transforma o agente em curador do próprio ecossistema. Você pede "ache uma skill para X" e ele resolve. É o ponto de entrada de tudo.

instalar:

$ npx skills add vercel-labs/skills

💡 Por que 1,8M e não 488k?

find-skills tem o dobro de installs da segunda colocada porque é instalada como pré-requisito de quase todo fluxo de descoberta. Vantagem de "chegou cedo e virou infraestrutura" — exatamente o viés de install count que a Trilha 1.5 vai dissecar.

3

🎨 O trio de frontend

Web & Frontend é o segundo maior grupo do catálogo, e três skills dominam: as duas referências de design da vercel-labs e a anthropics/skills. Mesmo que você não seja dev de front, ler elas mostra como escrever uma skill de "guideline" — instruções de estilo que o agente aplica sem você repetir.

frontend-design · anthropics/skills

488.299

O que faz: gera interfaces polidas, com gosto, fugindo do visual genérico de IA. No pack: é a skill da própria Anthropic — referência de como escrever uma skill de design e gatilho de qualidade visual imediata.

vercel-react-best-practices · vercel-labs/agent-skills

443.261

O que faz: codifica as boas práticas de React/Next que a Vercel recomenda — composição, server components, padrões de dados. No pack: melhora qualquer código React que o agente escreva e mostra como condensar uma stack inteira num SKILL.md.

web-design-guidelines · vercel-labs/agent-skills

358.627

O que faz: diretrizes de layout, tipografia, espaçamento e acessibilidade aplicadas a qualquer UI. No pack: par perfeito com frontend-design — uma cuida do "como construir", a outra do "como deve parecer".

instalar o trio:

$ npx skills add anthropics/skills
$ npx skills add vercel-labs/agent-skills   # traz react + design guidelines
4

🤖 As duas de agentes: skill-creator e agent-browser

Duas skills do grupo AI/Agents merecem entrar cedo porque mudam o que o seu agente consegue fazer. Uma ensina o agente a criar novas skills; a outra dá a ele um navegador.

skill-creator

246k

anthropics/skills

Faz: guia o agente a escrever, revisar e otimizar SKILL.md. No pack: é a fonte canônica de "como uma skill deve ser" — você vai usar ela na Trilha 1.4 para criar a sua.

agent-browser

332k

vercel-labs

Faz: automatiza navegação, preenche formulários, tira screenshots, extrai dados de páginas. No pack: destrava casos de uso reais (testar UI, raspar dados) com uma instalação só.

instalar:

$ npx skills add anthropics/skills      # skill-creator
$ npx skills add vercel-labs/agent-skills  # agent-browser

💡 Skills do mesmo repo vêm juntas

Vários nomes do pack moram no mesmo repositório (anthropics/skills traz frontend-design + skill-creator; vercel-labs traz várias). Um npx skills add owner/repo pode instalar um pacote inteiro de uma vez.

5

🗄️ Dados e processo: supabase + TDD

Para fechar o pack com algo além de front, duas skills de domínios diferentes mostram a amplitude do catálogo: uma de dados (a líder do grupo Data) e uma de processo de engenharia (a referência da comunidade superpowers).

supabase-postgres-best-practices · supabase

203k

O que faz: ensina o agente a escrever SQL, RLS e schema Postgres do jeito que a Supabase recomenda. No pack: é a skill mais instalada de Data & Databases e um ótimo exemplo de skill de fornecedor (vendor) feita pelo próprio time do produto.

test-driven-development · obra/superpowers

107k

O que faz: faz o agente escrever o teste antes do código, em loop red-green-refactor. No pack: melhora a qualidade de QUALQUER tarefa de código e é o exemplo canônico de skill de "processo" — não dá ferramenta, muda o comportamento.

instalar:

$ npx skills add supabase/supabase
$ npx skills add obra/superpowers

💡 Três sabores de skill no pack

Repare que o starter pack cobre os três tipos que você vai reencontrar no curso: meta (find-skills, skill-creator), guideline/vendor (frontend, react, supabase) e processo (TDD). Conhecer um de cada já te dá vocabulário para o resto.

6

📦 O kit mínimo recomendado

Não precisa instalar os oito de uma vez. Comece pelo kit mínimo de 3 — o essencial para qualquer pessoa — e expanda conforme seu trabalho. Aqui está a ordem recomendada, em passos.

1

Kit mínimo (todo mundo): find-skills + skill-creator

Com descoberta + criação você já consegue achar tudo o que existe e criar o que falta. É o par fundacional. Adicione test-driven-development se você escreve código.

2

Se faz front: + frontend-design + web-design-guidelines

O par de design da Anthropic e da Vercel cobre "construir" e "parecer bom". Adicione vercel-react-best-practices se a stack for React/Next.

3

Por necessidade: agent-browser e supabase

agent-browser quando precisar automatizar a web; supabase-postgres-best-practices quando o projeto tiver Postgres. Instale sob demanda — não acumule skills que você não usa.

o kit mínimo em dois comandos:

$ npx skills add vercel-labs/skills   # find-skills
$ npx skills add anthropics/skills    # skill-creator + frontend-design

💡 Menos é mais

Cada skill instalada gasta um pouco do contexto do agente (o name + description vive sempre carregado). Um pack enxuto e relevante funciona melhor do que 40 skills concorrendo pela atenção. Instale o que usa; remova o que não usa.
